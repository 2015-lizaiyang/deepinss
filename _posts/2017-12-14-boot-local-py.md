## 整体流程

SS 分为 local 端 和 server 端, 我们的请求发给 local 端, local 端会加密传递给 server 端, server 端帮助我们发出真正发给目标服务器的请求, 目标服务器将数据传递给 server 端, server 端加密数据回传给 local 端, local 端将数据解密
